Blog Update

Updating my blog

The reaction to my blog from people whom I’m close with were pretty similar. They all tended to think it wasn’t very polished and could use some more work around the edges. They enjoyed reading it, but they felt that some of my technical post with code samples could use a background that was different from the background of the normal text.

I followed up and worked changed a lot of the CSS to make it more dynamic. I had originally wrapped my entire site within a div to control the resolution. This created some issues with the divs inside the wrapper not lining up correctly. I deleted this and the divs line up much better and makes the site look much smoother. I also added a container to house the samples of code on my technical blog, which does make it look a lot cleaner and easier to read.

I’ve enjoyed blogging throughout Phase 0. I’ve learned a lot of different things that I never expected to learn from DBC. It also serves as a mental exercise with topics that are technically and culturally related.

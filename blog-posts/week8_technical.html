Recursion

Recursion is when a method will call itself as long as the defined base case is not met. A popular example of a recursive method is to create a factorial function. To start, the mathematical function of a factorial is:

n! = 1 (when n==0)
(otherwise)
= n * (n-1)!

If I were to use the above function, it would look like this:

5! = 5 * 4 * 3 * 2 * 1 = 120

In order to get the factorial of 5 using recursion, we could write method in Ruby like this:

def factorial(n)
if n == 0 # => defined base
return 1
else
factorial(n-1) * n # => calling itself as long as the base isn't met
end
end

puts factorial(5)

=> 120

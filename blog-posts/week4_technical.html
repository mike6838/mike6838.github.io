Week 4 - Technical Blog

Enumerable#map

The .map method is a very convenient built in Ruby method that allows you to make an exact copy of the object that is being called on. I find myself most often using the map method when iterating over arrays. Since the map method makes a copy of the object, the output gives us the data after it's been manipulated, where as a method like #each would give us the original data.

Here's an example,

001 >[2, 4, 6].map { | i | i * 2 }
=> [4, 8, 12]

Iterating over the same data using the each method, instead of map, would give us the original data.

Like so,

002 >[2, 4, 6].each { | i | i * 2 }
=> [2, 4, 6]

In conclusion, the map method is a very useful and convenient method to use when manipulating data.
